# thistime.github.io

Minimal GitHub Pages landing site for **ThisTime**, a startup concept that pairs a focused desk countdown device with a companion mobile app for deadline awareness, task focus, and procrastination support.

## Local preview

Open `index.html` in your browser, or serve the folder locally:

```bash
python3 -m http.server 8000
```

Then visit <http://localhost:8000>.

## Deploy on GitHub Pages

Enable GitHub Pages in repository settings to publish from the default branch root. User/organization sites use a `username.github.io` repository name, while project sites can use any repository name.
